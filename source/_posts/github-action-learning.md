# 确定需求

之前也说过，我的博客是使用 Hexo 进行搭建的，传统方法是编辑好文章后手动进行一次生成，再手动推送到 GitHub 对应的仓库，之后托管的网页才能上线。虽然编辑文章可以在多种平台上完成，但成品网页的生成需要依赖 Node.js，至少在移动端设备上支持有限，比较麻烦。

解决办法很简单，把网页生成这一步交由云端服务来实现即可。
