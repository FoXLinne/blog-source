# 前言
之前的博客是托管在 `GitHub Pages` 上的，虽然部署和维护都比较方便，但因为皆知原因，`GitHub Pages` 的访问速度和稳定性在国内并不理想。为了提升博客的访问体验，我决定将博客部署到自己的服务器上。
